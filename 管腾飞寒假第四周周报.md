## 进入正题

<details>
  <summary><b>第一步连接网络</b></summary>

  
```
//查看ip
ip a

//进入iwctl界面 
iwctl

//查看设备
device list「输出：wlan0」

//指定设备扫描网络
station wlan0 scan

//列出能连接的网络
station wlan0 get-networks

//连接网络
station wlan0 connect （网络名）

//退出
exit

//测试 -c为指定次数，即指定测试3次，若不写可以使用ctrl+c终止
ping -c 3 bilibili.com

```


</details>

+ 主要使用自带工具iwctl

<details>
  <summary><b>时间调整</b></summary>

```
//确认开启NTP【网络时间协议】
timedatectl

//若未开启
timedatectl set-ntp true

```


</details>

<b>镜像文件</b>
```
//reflector为工具，-a为近12小时更新过的源 -c cn指定中国 -f 10 选择最快的10个 --sort score为同步时间和下载速度的排序
//--v 显示过程 --save /etc/pacman。。。将结果保存到这个文件夹里
reflector -a 12 -c cn -f 10 --sort score --v --save /etc/pacman.d/mirrorlist
```
对于linux，镜像源就是欧美等地方软件的副本

若没有镜像源下载速度会变得非常慢

**数据库密钥**
```
//安装密钥 -S为安装，-y为更新软件数据库，pacman -Sy不能直接使用
//配合archlinux-keyring是少数安全的指令，一般都为pacman -Syu
pacman -Sy archlinux-keyring
```
