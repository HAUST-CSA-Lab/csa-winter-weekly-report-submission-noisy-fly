根据教程成功在电脑中配置arch  

以下为详细步骤与对步骤的理解  

## 准备工作

-第一步为制作启动盘，这里使用的是ventoy

-对于原本为Windows系统的电脑来说， 首要步骤应当为磁盘分区

## 进入正题

<details>
  <summary><b>第一步连接网络</b></summary>

  
```
//查看ip
ip a

//进入iwctl界面 
iwctl

//查看设备
device list「输出：wlan0」

//指定设备扫描网络
station wlan0 scan

//列出能连接的网络
station wlan0 get-networks

//连接网络
station wlan0 connect （网络名）

//退出
exit

//测试 -c为指定次数，即指定测试3次，若不写可以使用ctrl+c终止
ping -c 3 bilibili.com

```


</details>
-主要使用自带工具iwctl

<details>
  <summary><b>时间调整</b></summary>

```
//确认开启NTP【网络时间协议】
timedatectl

//若未开启
timedatectl set-ntp true

```


</details>
